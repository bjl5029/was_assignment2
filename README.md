## About
웹 서비스 설계 과제 2
리액트를 기반으로 한 넷플릭스 클론 코딩.

## Install

Install [node.js](http://nodejs.org/)

Install dependencies, as specified in package.json

```bash
npm install
```

## Build & Run

```bash
npm start
``` 
* start a web server at localhost:3000. 

Go to http://localhost:3000/was_assignment2
to test the examples.
